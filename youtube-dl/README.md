youtube-dl
==========

[`youtube-dl`][1] is a small command-line program to download videos from YouTube.com and a few more sites.

## Tutorial

```
# create a alias
$ alias yt='docker run --rm -v `pwd`:/data vimagick/youtube-dl'

# download a video
$ yt nVjsGKrE6E8
```

[1]: https://rg3.github.io/youtube-dl/
