hydra
=====

A very fast network logon cracker which support many different services.

- [x] openssl
- [x] idn
- [x] curses
- [x] pcre
- [x] Postgres
- [ ] SVN
- [ ] firebird
- [x] MYSQL client
- [ ] AFP
- [ ] NCP
- [ ] SAP/R3
- [ ] libssh
- [ ] Oracle
- [ ] GUI req's
- [ ] Android specialities
- [ ] secure compile option support in gcc
