[`Shadowsocks`][1] is a secure socks5 proxy,
designed to protect your Internet traffic.

> If you want to keep a secret,
> you must also hide it from yourself.

## docker-compose.yml

```
shadowsocks:
  image: vimagick/shadowsocks:latest
  ports:
    - "8388:8388"
  environment:
    - PASSWORD=secret
  restart: always
```

## server

```
$ docker-compose up
```

## client

```
$ sslocal -c config.json
```
