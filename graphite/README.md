graphite
========

[Graphite][1] does three things:

- Kick ass.
- Chew bubblegum.
- Make it easy to store and graph metrics.

## docker-compose.yml

```yaml
graphite:
  image: vimagick/graphite
  ports:
    - 2003:2003
    - 2004:2004
    - 7002:7002
    - 8080:8080
    - 9001:9001
  volumes:
    - ./data:/opt/graphite/storage
  restart: always
```

## Up and Running

```bash
$ cd ~/fig/graphite
$ mkdir -p data/log/webapp
$ docker-compose up -d
>>> python webapp/manage.py migrate --run-syncdb --noinput
>>> exit
$ tree -F -L 2
├── data/
│   ├── carbon-cache-a.pid
│   ├── graphite.db
│   ├── log/
│   └── whisper/
└── docker-compose.yml
$ curl http://localhost:8080
```
